# 📝 To-Do List App

A simple To-Do List web application built using HTML, CSS, and JavaScript.

## 🚀 Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Tasks saved using Local Storage

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📂 How to Use
1. Open the project folder
2. Open `index.html` in browser
3. Start adding your tasks

## 🎯 Learning Purpose
This project is created to practice DOM manipulation and JavaScript basics.
